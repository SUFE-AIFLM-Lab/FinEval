# 如何提交你的测评

您首先需要准备一个UTF-8编码的JSON文件，并按照以下格式编写。
```
## 每个学科内部的键名是数据集中的"id"字段
{
    "banking_practitioner_qualification_certificate": {
        "0": "A",
        "1": "B",
        "2": "B",
        ...
    },
    
    "学科名称":{
    "0":"答案1",
    "1":"答案2",
    ...
    }
    ....
}
```

然后你可以将准备好的JSON文件提交到zhang.liwen@shufe.edu.cn。
